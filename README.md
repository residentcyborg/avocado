# Avocado

A Forth for Linux and macOS on x86-64.

Avocado is written mostly in (direct-)threaded code. There is no limit on the amount of primitives, however, each primitive should be kept small.

## Prerequisites

* `make`
* `gcc` or `clang`

## Usage

Build:

	make

Run:

	make run

Quit:

	ctrl^d

Clean:

	make clean
